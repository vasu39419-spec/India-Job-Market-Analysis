## 🎯 Business Context

### The Problem I Personally Faced

As a B.Com student at Sri Aurobindo College Delhi preparing 
for placements, I spent 2 weeks manually researching jobs 
across multiple portals trying to answer basic questions:

- Which industries pay freshers the best salary?
- Should I target Bangalore or Delhi NCR?
- Are Unicorns better than MNCs for freshers?
- What is a realistic salary expectation for 0–1 year experience?

The information was **fragmented, inconsistent, and time-consuming** 
to find. So I treated my own placement challenge as a 
**Business Analysis case study**.

### The Solution I Built

An end-to-end BA project that includes:

✅ Stakeholder analysis and requirements gathering  
✅ Business Requirements Document (BRD)  
✅ Agile user stories tracked in Jira  
✅ Process flow mapping (As-Is vs To-Be)  
✅ SQL analysis in PostgreSQL  
✅ 4-page interactive Power BI dashboard  

### Business Impact

| Metric | Before | After | Improvement |
|--------|--------|-------|-------------|
| Research Time | 14 Days | 30 Minutes | 95% reduction |
| Jobs Analyzed | ~50 manual | 5,247 | 100x more |
| Decision Confidence | 60% | 95% | +35% |
| Data Source | Multiple portals | Single dashboard | Centralized |

## 📈 Key Business Findings

✅ **IT leads hiring volume** — 35% of all job listings  
✅ **FinTech pays highest for freshers** — Avg ₹8.5 LPA  
✅ **Bangalore dominates** — 28% of total job opportunities  
✅ **On-site work still leads** — 52% vs 38% hybrid vs 10% remote  
✅ **Unicorns pay premium** — ~15% higher salary than MNCs  
✅ **Salary range** — ₹0.8 LPA to ₹115.4 LPA (avg ₹19.83 LPA)

## 🗺️ Process Flow: As-Is vs To-Be

### Before Dashboard (As-Is)


Student → Opens Naukri → Opens LinkedIn → Opens Glassdoor
→ Conflicting Data → Manual Excel → Confused Decision
Time: 14 Days | Confidence: 60%


### After Dashboard (To-Be)
Student → Opens Dashboard → Selects Filters
→ Views KPIs → Compares Salaries → Exports Shortlist
→ Confident Decision
Time: 30 Minutes | Confidence: 95%
